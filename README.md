# Toast Message 과제
  - Toast Message 직접 구현
    - 위치 지정 가능
    - Delay 설정 및 auto close 가능
    - Delay가 null일 경우 auto Close X
    - 닫기 버튼 가능
  - All Clear 구현
  - Progress bar 구현

# 사용 기술
  - React (TypeScript)
  - Recoil (전역 상태관리)
  - styled-componetns (스타일링)

# 실행 방법
  - VS code로 폴더 Open
  - npm install
  - npm start
